# 使用Express+MongoDB搭建多人博客

### 版本
**Node.js**: 0.10.32<br>
**Express**: 4.13.1

### 功能
 1. 多人注册、登陆
 2. 发表文章(markdown)
 3. 上传文件
 4. 文章的编辑与删除
 5. 存档
 6. 标签
 7. 分页
 8. 留言(markdown)
 9. 用户个人主页
 10. 文章pv统计及留言统计
 11. 用户头像
 12. 标题关键字查询(支持有限的正则查询)
 13. 友情链接
 14. 404页面
 15. 转载功能
 16. 日志功能
